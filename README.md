# streaming-studio

A streaming application by electron.

## Getting Started

### 1. Resolve dependencies

```shell
yarn install
```

### 2. Build sources

```shell
yarn build
```

### 3. Run application

```shell
yarn start
```

### 4. Check lint before commit

```shell
yarn lint
```

and fix if lint failed.

```shell
yarn lint:fix
```

## Packaging

```shell
yarn package
```
